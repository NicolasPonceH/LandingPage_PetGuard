# 📖 Contextualización

**PetGuard** es un innovador sistema inteligente para el monitoreo de la salud de mascotas.

Esta Landing Page fue desarrollada para presentar de manera atractiva e interactiva todo el ecosistema PetGuard, incluyendo:

- 🐕 Collar Inteligente (Hardware Premium)
- 📊 Dashboard de monitoreo en tiempo real
- 📱 Aplicación móvil
- 🏥 Integración con clínicas veterinarias
- ❤️ Sistema de alertas inteligentes

La página utiliza una narrativa visual moderna para mostrar cómo PetGuard ayuda a los dueños a cuidar mejor la salud de sus mascotas mediante métricas biométricas y monitoreo continuo.

# ⚙️ Instalación

## 1️⃣ Clonar el repositorio

```bash
git clone https://github.com/NicolasPonceH/LandingPage_PetGuard.git
```

## 2️⃣ Ingresar al proyecto

```bash
cd LandingPage_PetGuard
```

## 3️⃣ Instalar dependencias

```bash
npm install
```

## 4️⃣ Ejecutar el proyecto

```bash
npm run dev
```

## 5️⃣ Abrir en el navegador

```
http://localhost:3000
```
